Security Gateway

This demo mimics a Cloudflare-style waiting room. The site has a hard cap on active sessions, a rolling anomaly detector for sudden entry-rate spikes, and a bounded local simulator instead of unrestricted traffic.

Open Admin Console

How this demo works

  1. The visitor asks for a slot.
  2. The server checks the active-session cap.
  3. If space is available, access is granted for 60 seconds.
  4. If capacity is full, the visitor sees the waiting page.
  5. The admin console can start a bounded local traffic simulation.
